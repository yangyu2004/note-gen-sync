# 03-JVM虚拟机

灵魂三问：**JVM是什么？**

* JVM广义上指的是一种规范。狭义上指的是JDK中的JVM虚拟机。

**为什么要学习JVM？**

1. 面试过程中经常会被问到JVM
2. 研发过程中肯定会面临一些重难点问题与JVM有关系。例如：线程死锁、内存溢出、项目性能优化等等。
3. 基础不牢，地动山摇。想要深入掌握Java这门语言，JVM始终是绕不过去的那座大山。

**怎样学习JVM？**

VM虚拟机部分我们是这样安排的：

1. 类加载子系统
2. 运行时数据区
3. 一个对象的一生（出生死亡与内涵）
4. GC垃圾回收器
5. JVM调优相关的工具与可调的参数
6. 调优实战案例

# JVM虚拟机概述

## 1.1 JVM基本常识

### 什么是JVM？

平常我们所说的JVM广义上指的是一种规范，狭义上指的是JDK中的JVM虚拟机，JVM的实现是由各个厂商来做的比如现在流传最广的是hotspot。

Java技术体系主要包括如下四个方面:

1. Java程序设计语言
2. Java类库API
3. 来自商业机构和开源社区的第三方Java类库
4. Java虚拟机:各种硬件平台上Java虚拟机的实现
